# Java Chat Application

## Overview
This project is a client-server based chat application developed using Java. It enables real-time communication between multiple clients connected to a central server.

## Features
- Real-time messaging
- Multi-client support
- Server-client architecture
- Efficient communication using sockets

## Technologies Used
- Java
- Socket Programming
- Multithreading

## Working
The server listens for incoming client connections. Multiple clients can connect to the server and exchange messages in real-time. Each client runs on a separate thread, allowing simultaneous communication.

## Applications
- Messaging systems
- Client-server communication models
- Networking practice
